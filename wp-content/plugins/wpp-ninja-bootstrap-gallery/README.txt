Turns your WordPress Gallery into a Bootstrap Responsive

== Description ==

This plugin turns the WordPress Gallery into a responsive one using Twitter Bootstrap. It uses the Magnific Popup to open the images in a responsive lightbox.

* Fully responsive across desktop, tablet and mobile devices
* No need to learn any new functionality for adding galleries to your website
* Supports existing caption functionality inside of WordPress
* Best of all, it's 100% Bootstrap
* Uses jQuery Magnific popup for the lightbox

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wppninja-bootstrap-gallery` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add a gallery to your Post / Page / or custom post like you normally would
4. Let the magic begin!

== Frequently Asked Questions ==

= Do I need Twitter Bootstrap pre-installed in my theme for this to work? =
Yes, you will need it to be installed.

= Can I change the default image size? =
Yes, you can, just select the alternative image size from the size drop down menu in the WordPress Gallery manager.

= I use a different lightbox, can I disable Magnific Popup? =
Unfortunately the Magnific Popup is hardcoded into the plugin. Please let me know if it causes any issues.
